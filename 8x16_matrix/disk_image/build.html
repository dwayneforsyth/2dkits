How to build Build

build source code unix

  • git clone https://github.com/dwayneforsyth/2dkits.git
  • cd 2dkits
  • git submodule update --init --recursive
  • esp-idf/install.sh
  • source esp-idf/export.sh
  • cd 8x16_matrix
  • idf.py build
  • idf.py -p [/dev/ttyUSBx] flash

build source code win10

  • git clone https://github.com/dwayneforsyth/2dkits.git
  • cd 2dkits
  • esp-idf/install.ps1
  • esp-idf/export.ps1
  • cd 8x16_matrix
  • idf.py build
  • idf.py -p [comxx] flash
